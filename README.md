# Berlin Nightlife App

A modern, map-first web application for exploring Berlin's club culture.

## Features
*   **Interactive Map:** Dark-mode map powered by Leaflet & CartoDB.
*   **Venue Profiles:** Detailed info for clubs like Berghain, Tresor, Sisyphos.
*   **Event Listings:** Mock data simulating Resident Advisor integration.
*   **Mobile-First Design:** "Citymapper-style" bottom sheet navigation.

## Tech Stack
*   **Frontend:** React, TypeScript, Vite
*   **Styling:** Vanilla CSS (Variables & Modules)
*   **Map:** Leaflet + React-Leaflet
*   **Icons:** Lucide React

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:5173](http://localhost:5173)
